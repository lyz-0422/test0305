Unit 01 JavaScript 基礎
JS · 01變數、型別、函式
宣告關鍵字

let 可重新賦值的變數；const 常數（物件內容仍可改）；var 舊版，有 hoisting 問題，現代開發避免使用。

let score = 95 const name = "LYZ" typeof 42 // → "number" typeof null // → "object" (歷史 Bug) `\${name} 的成績是 \${score} 分` // template literal
陣列方法鏈式操作

map() 轉換每個元素、filter() 篩選、reduce() 累積。可以鏈式組合使用。

[1,2,3,4,5] .filter(x => x % 2 === 0) // [2, 4] .map(x => x * 3) // [6, 12]

Unit 04 Vue 3 Tutorials
VUE · 01Vue 3 核心概念
為什麼選 Vue 3？

Vue 3 相較 Vue 2 引入了 Composition API，讓邏輯可以按「功能」組織而非按「選項」分散，搭配 TypeScript 支援更完善，打包體積更小。

特性Options API（Vue 2 風格）Composition API（Vue 3 推薦）
邏輯組織分散在 data/methods/computed按功能集中在 setup()
TypeScript支援有限原生支援
邏輯複用Mixins（易衝突）Composables（清晰無衝突）
SFC · Single File Component

Vue 的 .vue 檔案由三個區塊組成，清晰分離結構、邏輯、樣式。

<script setup> import { ref, computed, onMounted } from 'vue' const count = ref(0) const doubled computed(() => count.value * 2) </script>
