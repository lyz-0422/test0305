CSS · 03
CSS Layout
RWD 響應式設計
›

@media query 針對不同螢幕寬度套用樣式，搭配 rem / % / vw 等彈性單位，讓網頁適配各種裝置。

@mediabreakpointrem · vwmobile-first
/* Mobile First — 從小螢幕開始 */ .grid { grid-template-columns: 1fr; } @media (min-width: 640px) { /* tablet */ .grid { grid-template-columns: 1fr 1fr; } } @media (min-width: 1024px) { /* desktop */ .grid { grid-template-columns: repeat(3, 1fr); } } /* 彈性字體大小 */ h1 { font-size: clamp(2rem, 5vw, 4rem); }
rem相對於根元素字體大小
vw / vh視窗寬 / 高的百分比
clamp(min, val, max)響應式數值，限制上下限
Mobile First先寫手機版，min-width 擴大
模擬裝置切換

CSS Animation
›

@keyframes 定義動畫關鍵影格，animation 屬性控制時長、緩動曲線、延遲與重複，transition 實現 hover 效果。

@keyframesanimationtransitiontransform
@keyframes slideUp { from { opacity:0; transform:translateY(20px); } to { opacity:1; transform:translateY(0); } } .card { animation: slideUp .6s ease both; /* name | duration | timing | fill-mode */ } /* transition for hover */ .btn { transition: background .2s, transform .3s; } .btn:hover { transform: translateY(-3px); }
animation-fill-modeboth — 保留開始與結束狀態
cubic-bezier自訂緩動曲線
transformtranslate / rotate / scale — 不觸發 reflow
will-change提示瀏覽器預先優化
