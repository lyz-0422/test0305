# README

✨ 專案亮點
多頁面架構：包含首頁、詳細個人介紹與分類筆記頁面，結構清晰。

響應式設計：自動適配不同裝置螢幕尺寸，提供流暢的行動端與桌面端瀏覽體驗。

導航系統：頂部固定導航列，讓使用者能快速切換不同模組。

多媒體整合：支持圖片展示區塊，增強頁面的視覺吸引力。

📸 頁面說明
🏠 首頁 (index.html)
網站的進入點，包含精簡的歡迎詞、個人品牌標識，以及前往各子頁面的快速入口。

🧑‍💻 關於我 (about.html)
詳細的個人介紹頁面，包含：

學經歷背景：系統化展示過往的學習軌跡。

專業技能：條列式呈現核心競爭力。

聯絡資訊：提供便捷的聯繫管道。

📝 學習筆記 (notes.html)
專為知識管理設計的頁面，用於紀錄技術點、專案心得與日常學習內容。

🛠 技術棧
HTML5：使用語義化標籤（如 <header>, <nav>, <main>, <footer>）確保 SEO 與無障礙訪問。

CSS3：實現現代化的佈局管理與頁面美化。

Vanilla CSS：無須依賴外部框架，確保極速的加載表現。
