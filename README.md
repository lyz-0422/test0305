# README

🎯 專案目標

個人品牌展示：透過 about.html 展示個人背景與技術棧。

知識管理：利用 notes.html 記錄學習筆記與開發心得。

良好的使用者體驗：建立直觀的導航系統，讓訪客能輕鬆切換不同頁面。

📂 頁面結構說明

本專案由以下核心頁面組成：

🏠 首頁 (index.html)

網站的入口點。

包含導航欄與主要視覺區域，引導使用者進入其他功能區塊。

🙋 關於我 (about.html)

詳細介紹開發者的背景、興趣與聯繫方式。

建立專業形象的窗口。

📝 技術筆記 (notes.html)

系統化地整理技術知識、代碼片段或開發日誌。

方便日後查閱與知識沉澱。

🛠 技術棧

Markup: HTML5 (結構化語義標籤)

Styling: CSS3 (佈局與視覺設計)

Deployment: GitHub Pages (預計部署環境)
